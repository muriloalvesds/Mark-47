Instalar node
Dependencias:
npm install

Rodando o sistema:

Alimentar a url no arquivo payload.json, ou alterar a chamada do sistema como no comando a baixo 
com o arquivo json contendo os paramentros de URL.
O sistema suporta varia entradas de url atraves de JSON

Executar o comando a baixo via terminal na raiz do projeto
node index.js payload.json 

Os resultados retornam diretamente no terminal
